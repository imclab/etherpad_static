Rubric:
  • Writes engaging blog posts
  • Writes clearly and concisely
  • Includes thoughtful and creative presentation of materials

  • Helpful Feedback Badge [all]
Description: The Helpful Feedback Badge recognizes the ability to provide helpful and interesting feedback while maintaining a respectful tone.

  • Provides helpful and interesting feedback.
  • Includes suggestions and simulates interesting conversation
  • Maintains a respectful tone.
  • Provides extra information, through external links and comparisons to other pieces of work.

Requirements

You need at least 1 peer to say that you provide helpful and interesting feedback while maintaining a respectful tone. 11

## Skill
Assessment type = formula

[One skill badge per challenge for 1.5]

  • HTML Basic Badge [4]
Description: The HTML Basic Badge recognizes the ability to create an html page by making basic use of HTML tags.
Assessment type: course organizer/mentor assessment, peer assessment?
Assessment formula: awarded once from course organizer? if accumulative then formula could be: course organizer/mentor + 3 peers
Rubric:
The peer is able to:
  • use the Mozilla Developer Network HTML reference [0] to identify and use basic HTML tags 
  •  translate HTML tags like <div>, <p> into real world structures

Badge Requirements text:
To get this badge you need 3 peers or 1 mentor to say that you are able to use the Mozilla Developer Network HTML reference [0] to identify and use basic HTML tags  and that you translate HTML tags like <div>, <p> into real world structures

  • DNS Basic Badge [5]
Description: The DNS Badge recognizes the ability to describe what DNS is and how it works.
Assessment type: formula
Rubric:
the peer is able to:
  • communicate what DNS is by using  different types of media
  • have a domain name that can be used to further explore DNS

Badge Requirements text:
To get this badge you need 3 peers or 1 mentor to say that you are able to communicate what DNS is by using different types of media and that you have a domain name that can be used to further explore DNS

  • Hosting Badge [6]
Description: The Hosting Badge recognizes the ability to implement web hosting services to set up your own domain.
Assessment type: formula
Rubric:
the peer is able to:
  • configure a DNS to point at a personal hosting account
  • provide valuable information on web hosting services

Badge Requirements text:
To get this badge you need 3 peers or 1 mentor to say that you are able to configure a DNS to point at a personal hosting account and that you provide valuable information on web hosting services

  • Publishing Badge  [7]
Description: The Publishing Badge recognizes the ability to take content from a local computer, publish it live on the web and be able to view it in a browser.
Assessment type: formula
Rubric:
the peer is able to:
  • install an FTP or SFTP client
  • upload content that is visible live in a browser

Badge Requirements text:
To get this badge you need 3 peers or 1 mentor to say that you are able to install an FTP or SFTP client and that you upload content that is visible live in a browser
