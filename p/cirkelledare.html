What information is necessary for badge creation? 

  • Url of the challenge associated with this badge
  • Badge Name: Cirkelledare (Study group leader)
    • max 225 characters
    • recommended max 128 characters if you want the full name to be sent to beta.openbadges.org
  • Badge's Short Description: Du är nu redo att skapa egna studiecirklar (You are now ready to create a study group of your own)
    • max 225 characters
    • recommended max 128 characters if you want the full short description to be sent as the badge description to beta.openbadges.org
  • Badge's Requirements: När du genomfört utmaningen Skapa en distanscirkel (When you have fulfilled the challange "Skapa en distanscirkel")
    • rich text
    • explain what is needed to obtain the badge
    • do not try to fit everything into the short description since this field allows you to further explain what this badge is about
  • Image:
    • the file extenssion needs to be .png and maximum size of 256kb to be compatible with beta.openbadges.org
    • Recommended dimenssions: width="150" height="150"
  • Badge Logic: Challenge Completion Badge (Stealth)
    • Choose one of:
      • Skill Badge
        • requires submissions
        • minimum N number of votes to be awarded (default and preferably 3 or 2)
        • minimum M average rating (default and preferebly 3)
        • can be associated with a task of the challenge so the user is promt to submit to the badge while (s)he is marking the task as completed
      • Community Bagde
        • users give this badge to their peers
        • appears in the give badge dialog on discussion areas of the challenges
        • users can receive multiple badges of this comming from different peers
        • no submissions
      • Submission-Enabled Community Badge
        • Extends Community Badge to also allo users to optionally post a submission that others can accept
      • Challenge Completion Badge (Stealth)
        • given automatically to anyone that marks all the challenge tasks as completed
      • Other stealth badges (not currently implemented / many of the stack overflow and osqa badges work like this)
        • can be issued based on user activity without user intervention, e.g. number of posts / comments 
      • If these option do not fit what you have in mind or you have questions lets us know
        • there are a few options we can change to allow different logics (e.g. add another badge as prerequisite)
  • (if it is an skill badge)
    • Rubrics: This is not relevant for this badge
      • A few sentences starting with 'How well was this peer able to: ' that reviewers will use to evaluate the submissions
    • Task (optional): https://p2pu.org/en/groups/skapa-en-distanscirkel/
      • url or the task associated with this badge
