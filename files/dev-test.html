The script should take less than 10 minutes to complete.

1. SSH onto dev.p2pu.org
2. Run '/etc/p2pu/setup_dev.sh'

3. It'll ask you for three things initially:
    a) The name of the new site, so something like 'sandbox' - the site will be accessed at thisname.dev.p2pu.org and is used in other places as well.
    b) Your SSH username for the LIVE p2pu.org site.
    c) The mysql root password for dev.p2pu.org (starts with 3du ...
    
The script will then begin to execute. It is broken down into several parts although you'll generally want to answer yes to all sections.

It will also ask you for your SSH password for the live site twice, as it retrieves information from www.p2pu.org.

All amendments to this document are welcome.
