We are trying to find a good tool for P2PU to make a collaborative bibliography of research on peer/open learning.

Stian's criteria:
- easy to add new publications by many people
- possible to import list of publications from some standard formats (if we already have lot's of relevant publications in our reference management, shouldn't have to retype it)
- tagging
- ability to export in a machine readable format (BibTex or similar) so we can download a bunch of references, and import to our local reference manager
- ability to "embed" list of publications in nice format on for example the P2PU wiki 

Options:
  • Mendeley
  • CiteULike
  • Zotero: allows to reference any content, primarily articles, but also web pages, documents, etc
    • I second the vote for Zotero. They now support group accts, so anyone can add a paper to a common folder from anywhere. It still does not do as good a job as I would like auto-scraping the relevant bibliographic information from web pages and pdfs, but then none of the alternatives do well either.
    • I also think that Acawiki is the clear choice for sharing and building on summaries ofpapers.
